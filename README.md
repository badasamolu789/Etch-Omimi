# ETCH

A premium creative marketplace landing experience with polished UI, SEO-friendly metadata, and a Supabase-ready architecture.

## Project structure
- index.html, about.html, ... — page entry points
- styles/main.css — shared styles and layout system
- scripts/main.js — shared interactions
- scripts/supabase.js — Supabase helper module
- assets/logo/ — brand assets
- docs/supabase-plan.md — integration checklist

## Next steps
1. Create a Supabase project.
2. Add your Supabase URL and anon key to the page via window.ETCH_SUPABASE_URL and window.ETCH_SUPABASE_ANON_KEY.
3. Load the Supabase client script from your project.
4. Replace the demo form feedback with real Supabase submissions.
